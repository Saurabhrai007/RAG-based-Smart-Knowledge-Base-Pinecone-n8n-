Project 2: RAG-based Smart Knowledge Base (Pinecone + n8n)

Description: A high-context AI agent that can answer complex business queries by retrieving data from a specialized knowledge base (SOPs, PDFs).

🟢 Key Features:

Persistent Memory: Uses Pinecone Vector Database to store and retrieve document embeddings.

Document Intelligence: Processes unstructured data (PDFs/Docs) and converts them into searchable vectors.

Contextual Accuracy: Prevents AI hallucinations by grounding responses in verified internal documentation.

Tech Stack: n8n, OpenAI/Claude API, Pinecone, Python (for data cleaning).

Business Impact: Provides instant, accurate support for internal teams, reducing the need for constant manual lookups.

🔺 Installation Guide:
▫ Import the provided JSON in n8n.
▫ Setup your API keys (Claude/Pinecone).
▫ Activate the workflow.
